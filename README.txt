DULCINEA · CORRECCIÓN FINAL DE IMÁGENES

Esta versión corrige la relación entre cada servicio y su fotografía.

PASOS
1. Descomprima este ZIP.
2. En GitHub > dulcinea-web > Add file > Upload files.
3. Suba TODOS los archivos.
4. Seleccione “Commit directly to the main branch”.
5. Pulse Commit changes.
6. Espere 1 a 3 minutos y abra Netlify en modo incógnito.

IMPORTANTE
Reemplace index.html y service-worker.js. Todos los archivos deben quedar en la raíz.
