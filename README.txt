CÓMO ACTUALIZAR DESDE EL CELULAR
1. Descomprima este ZIP.
2. Abra GitHub y entre al repositorio dulcinea-web.
3. Pulse Add file > Upload files.
4. Seleccione TODOS los archivos de la carpeta descomprimida.
5. Elija “Commit directly to the main branch”.
6. Pulse Commit changes.
7. Espere entre 1 y 3 minutos.
8. Abra https://dulcinea-tipografia.netlify.app
9. Si aparece la versión anterior, actualice la página o borre la caché del navegador.

IMPORTANTE
Todos los archivos deben quedar visibles en la raíz del repositorio.
No cree carpetas nuevas para las imágenes.
